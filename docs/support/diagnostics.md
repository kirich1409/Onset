# Диагностика Onset

## Экспорт журнала событий

Если вы столкнулись с проблемой и хотите отправить диагностику:

1. Нажмите на значок Onset в строке меню.
2. Выберите **«Экспортировать диагностику»**.
3. Выберите папку для сохранения файла (формат: `onset-diagnostics-ГГГГ-ММ-ДД-ЧЧММСС.log`).
4. Файл откроется в Finder автоматически.

Файл содержит записи журнала событий за последние 30 минут работы приложения.
Персональные данные в журнал не записываются.

## Отправка crash-логов (`.ips`)

При аварийном завершении Onset macOS сохраняет отчёт о сбое в папку
`~/Library/Logs/DiagnosticReports/`. Файлы называются по шаблону `Onset-*.ips`.

Чтобы приложить crash-лог к обращению в поддержку:

1. Откройте Finder.
2. Нажмите **Перейти → Перейти к папке…** (`⇧⌘G`).
3. Введите `~/Library/Logs/DiagnosticReports/` и нажмите **Перейти**.
4. Найдите файл `Onset-*.ips` с нужной датой и приложите его к описанию проблемы.

Также crash-логи доступны в приложении **Консоль**:
откройте **Консоль → Отчёты о сбоях → Onset**.
